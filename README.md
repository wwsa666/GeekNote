# 极客笔记 GeekNote

一个纯前端的本地笔记应用，数据存在浏览器里，不需要服务器，也不用注册账号。打开就能用。

---

## 为什么做这个

市面上的笔记应用要么需要登录，要么数据存在云端，要么界面太花哨加载半天。我就想要一个简单的东西：打开浏览器就能写，数据只留在本地，操作流畅不卡。所以就有了这个项目。

## 怎么用

1. 下载项目文件夹
2. 直接用浏览器打开 `index.html`（推荐 Chrome 或 Edge）
3. 没了

不需要安装任何依赖，不需要启动服务器，不需要 Node.js。纯 HTML/CSS/JS，打开即用。

## 功能介绍

### 📝 富文本编辑

选中文字后会弹出浮动工具栏，支持：

- **加粗** / *斜体* / <u>下划线</u>
- 文字颜色、背景色（内置色板，点击即用）
- 大号字 / 中号字 / 小号字
- 标题（H1、H2）
- 有序列表 / 无序列表
- 左对齐 / 居中 / 右对齐

标题区域也有独立的工具栏，可以单独调整标题的样式和颜色。

### 🖼️ 图片

支持三种方式插入图片：

- **粘贴** — 截图后直接 Ctrl+V 粘贴到笔记里
- **拖拽** — 从文件管理器拖图片进来
- **按钮** — 点工具栏上的图片按钮，选文件上传

图片插入后可以：

- **拖拽边缘调整大小** — 鼠标放到图片的右边缘、下边缘或右下角，出现双向箭头后拖拽即可缩放，可以放大超过原始尺寸
- **布局切换** — 选中图片后弹出布局面板，支持嵌入型、左环绕、右环绕和上下型四种排列方式
- **剪切** — 图片工具栏提供剪切按钮
- **删除** — 选中图片后 Delete/Backspace 删除，支持 Ctrl+Z 撤销恢复

图片以 Blob 存储在 IndexedDB 中，大于 1MB 的图片会自动压缩。

### 📁 文件夹

- 左上角点 📁 按钮新建文件夹
- 支持**嵌套文件夹**，可以在文件夹里再建子文件夹
- 点文件夹前面的箭头 ▶ 展开/收起
- 把笔记**拖到文件夹上**就能移进去
- 展开/收起状态会记住，刷新页面不会重置

右键文件夹可以：
- 重命名
- 删除（会提示确认）
- 在文件夹内新建子文件夹

### 🏷️ 标签

每条笔记可以打标签。点笔记标题下方的 ➕ 按钮添加标签，多个标签用逗号分隔。标签会显示在笔记列表的侧边栏里，搜索时也会匹配标签内容。

### 🔍 搜索

顶部搜索框实时过滤笔记列表，匹配范围包括标题、正文和标签。命中的关键词会高亮显示。在编辑区域也会标出搜索词的位置。

### 📌 排序

支持三种排序方式：
- 按修改时间
- 按创建时间
- 按标题

点旁边的 ↓ 按钮切换升序/降序。

### 🌙 主题切换

左上角月亮图标切换深色/浅色主题。默认深色主题，紫色系配色。浅色主题是柔和的灰白色调。选择会保存到 localStorage，下次打开自动恢复。

### 💾 自动保存

编辑内容后 800ms 自动保存到 IndexedDB，不用手动操作。滚动位置也会记住，切换笔记再回来不会丢失在位置。

### 📤 导入导出

- **导出** — 点 ⬇ 按钮，所有笔记、文件夹结构和图片会打包成一个 JSON 文件下载
- **导入** — 点 ⬆ 按钮选择之前导出的 JSON 文件，智能合并：
  - 相同 ID 的笔记保留更新时间更新的那个
  - 内容完全一样的笔记不会重复导入
  - 图片数据会一并导入

这样你就可以在不同浏览器之间迁移数据了。

### ✅ 批量操作

笔记列表里每条笔记前面有个复选框，勾选后可以批量删除。

### 🖱️ 右键菜单

在笔记列表和文件夹上右键，弹出自定义菜单，可以快速执行 置顶、重命名、删除等操作。

## 数据存储

所有数据存在浏览器的 IndexedDB 中，包括三个存储表：

| 存储表 | 内容 |
|--------|------|
| `notes` | 笔记数据（标题、富文本内容、标签、时间戳） |
| `images` | 图片 Blob 数据 |
| `folders` | 文件夹结构 |

### 容量

- Chrome / Edge：磁盘空间的 60% 左右，一般几十 GB 不成问题
- Firefox：单域名最大 2GB
- Safari：约 1GB（iOS 约 50MB）

存几千条笔记带几百张图片完全没问题。如果想查看当前用了多少空间，在浏览器控制台运行：

```javascript
navigator.storage.estimate().then(e => {
    console.log('已用:', (e.usage / 1024 / 1024).toFixed(2) + ' MB');
    console.log('配额:', (e.quota / 1024 / 1024 / 1024).toFixed(2) + ' GB');
});
```

### ⚠️ 注意

- **清除浏览器数据会丢失所有笔记**，建议定期导出备份
- Safari 有 7 天未访问自动清除的策略，不推荐在 Safari 上做主力使用
- 隐私模式下存储配额大幅缩减，不建议使用

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构，`contenteditable` 实现富文本编辑 |
| CSS3 | CSS 变量、Flexbox、动画、深浅主题 |
| Vanilla JS (ES6+) | 零依赖，不用任何框架 |
| IndexedDB | 本地数据持久化 |
| Blob API | 图片二进制存储 |

项目只有三个文件：

```
index.html   — 页面结构
style.css    — 样式
app.js       — 全部逻辑
```
